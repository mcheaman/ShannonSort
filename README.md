# ShannonSort
Modified insertion sort method that conceptualizes an array as 2d and sorts by rows  
 Ex. input = 13 14 94 82 25 59 94 65 23 45 27 73 25 39 10  
     initial gap size = 5  
     gap increase = 5  

First sort    
13 14 94 33 82  
25 59 94 65 23  
45 27 73 25 39   
10  

Second sort  
13 14 23 25 33 59 65 82 94 94   
6  

Third sort    
6 13 14 23 25 33 59 65 82 94 94
